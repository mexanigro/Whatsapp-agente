# Arzac Studio — Información del Negocio

## Qué es Arzac Studio
Arzac Studio es un servicio de suscripción mensual que ofrece a negocios locales en Israel una solución digital completa: web profesional personalizada + CRM + chatbot IA + agente de WhatsApp + mantenimiento continuo.

## Propietario
Liam Arzac | website@arzac.studio | arzac.studio

## Qué incluye el servicio (790 ₪/mes)

### 1. Web profesional personalizada
- Diseñada específicamente para el nicho del cliente (barbería, estética, tattoo, uñas, abogados, etc.)
- Se construye SIN preguntar al cliente — Liam investiga su Instagram, visita el local, fotografía el ambiente y crea la identidad visual completa
- Incluye: landing page, sistema de reservas/turnos, galería, testimonios, ubicación, horarios
- Multiidioma: hebreo, inglés, ruso
- Responsive y mobile-first
- Subdominio: [negocio].arzac.studio
- 5 variantes de splash screen animado

### 2. Micro CRM integrado
- Panel de administración para el dueño del negocio
- Gestión de turnos/reservas con validación server-side
- Base de datos de clientes
- Registro de clientes walk-in (sin reserva previa)
- Inbox de mensajes/consultas
- Gestión de staff (modo equipo)
- Métricas y análisis del negocio
- Email automático de confirmación y recordatorio 24h antes
- Pestaña de Soporte para pedir cambios, reportar problemas y hacer consultas

### 3. Chatbot IA 24/7
- Integrado en la web del cliente
- Conoce todo sobre el negocio (servicios, precios, horarios, ubicación)
- Responde preguntas de clientes en su idioma
- Nunca inventa información

### 4. Agente de WhatsApp IA
- Contesta mensajes y vende por el negocio 24/7
- Personalizado para cada negocio

### 5. Mantenimiento y soporte
- Agente monitor que supervisa el funcionamiento de las webs 24/7
- Capacidad de auto-reparación de errores
- Si no puede resolver, pide ayuda a Claude
- Soporte técnico continuo via pestaña de Soporte en el CRM (NO se hacen llamadas)
- Se pueden modificar imágenes, precios y branding completo cuando quieran

### 6. Hosting y dominio
- Hosting en Vercel (incluido en el precio)
- Dominio incluido en el precio
- Sin costos adicionales

## Proceso de venta
1. Liam identifica el negocio potencial
2. Construye la web ANTES de contactar al cliente (sin preguntarle)
3. Muestra la web con un tour interactivo guiado
4. El cliente ve su negocio ya digitalizado y decide
5. Firma contrato + primer pago → la web sale al aire en 48 horas

## Ecosistema técnico (3 sistemas)
1. **Master Template** — Web de cada cliente (React + Vite + Express + Firestore, deploy en Vercel)
2. **Nichos Hub** — Dashboard de operaciones de Liam (Next.js + Firestore + PostgreSQL, deploy en Railway)
3. **Monitor Agent** — Supervisor 24/7 de las webs (Node.js + Claude API, deploy en Railway)
